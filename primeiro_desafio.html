Tags I,U, STRONG, MARK, SMALL, SUB, SUP E BLOCKQUOTE

  • Quando utilizamos a tag I sigifica que o texto será escrito em itálico
  • Quando utilizamos a tag U Significa que o texto será escrito com sublinhado
  • Quando utilizamos a tag STRONG significa que o texto será escrito em negrito
  • Quando utilizamos a tag MARK Significa que o texto será marcado
  • Quando utilizamos a tag SMALL siginifica que este texto dentro da tag irá aparecer com tamanho menor
  • Quando utilizamos a tag SUB significa que o texto dentro esta tag ficará um pouco abaixo do restante do conteúdo
  • Quando utilizamos a tag SUP significa que o texto presente nela ficará um pouco acima
  • Quando utilizamos a tag BLOCKQUOTE
    significa que o texto é uma citação

    Podemos utilizar as tags FONT, DEL E ABBR

  • Utilizando a Tag DEL significa que este conteúdo foi excluida do texto
  • Utilizando a Tag FONT significa que você está utilizando um atributo para alterar a fonte
  • Utilizando a Tag ABBR significa que estaos utilizando uma abreviação
